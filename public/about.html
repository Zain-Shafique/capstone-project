About This Project

This AI Text Analysis Platform was created as a capstone project to showcase various text processing and analysis capabilities using Python and serverless functions.

Features

  • Sentiment Analysis - Analyze the emotional tone of text using TextBlob
  • Text Summarization - Generate concise summaries from longer texts using Sumy
  • Keyword Extraction - Extract the most important terms from your content
  • Content Enhancement - Improve readability with transition phrases
  • Language Translation - Translate text between multiple languages

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python serverless functions on Vercel
  • Libraries: TextBlob, NLTK, Sumy, Deep-Translator

How It Works

This application uses serverless functions to process text through various Python libraries. When you submit text for analysis:

  1. Your browser sends the text to the appropriate API endpoint
  2. The serverless function processes your text using the relevant Python libraries
  3. Results are returned to your browser and displayed in a user-friendly format
